init函数

程序中每个代码文件里的 init 函数都会在 main 函数执行前调用。这个 init 函数将标准库里日志类的输出，从默认的标准错误（stderr），设置为标准输出（stdout）设备。每个包可以有多个init函数，同一个包的init执行顺序，golang没有明确定义，编程时要注意程序不要依赖这个执行顺序

在 main 函数返回前，清理并终止所有之前启动的 goroutine。编写启动和终止时的状态都很清晰的程序，有助减少 bug，防止资源异常。

这个程序使用 sync 包的 WaitGroup 跟踪所有启动的 goroutine。非常推荐使用 WaitGroup 来
跟踪 goroutine 的工作是否完成。WaitGroup 是一个计数信号量，我们可以利用它来统计所有的
goroutine 是不是都完成了工作。

算法分析

时间负责度: O(nlogN)

空间复杂度: O(n)

简单嗦一下, 时间复杂度是一个函数，它定性描述该算法的运行时间。
常见时间复杂度 :

  • 常数阶 O(1)
  • 对数阶 O(logn)
  • 线性阶 O(n)
  • 常数对数阶 O(nlogn)
  • 平方阶O(n²)
  • 立方阶O(n³)
  • 指数阶(2^n)

空间复杂度是指算法执行程序并产生输出所需的内存
